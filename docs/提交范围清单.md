# 提交范围清单（Git 提交规则）

## must submit
- `backend/src/**`、`backend/pom.xml`
- `frontend/src/**`、`frontend/package.json`、`frontend/package-lock.json`、`frontend/index.html`
- `backend/src/main/resources/db/migration/**`
- `backend/src/main/resources/db/seed/seed_demo.sql`
- `docs/**`
- `README.md`
- `.github/workflows/ci.yml`
- `.gitignore`

## must not submit
- 构建产物：`backend/target/`、`frontend/dist/`、`frontend/.vite/`、`frontend/.cache/`
- 依赖目录：`frontend/node_modules/`
- 本地环境：`.env`、`.env.*`、`.venv*/`、`.python-version`
- 编辑器与系统垃圾：`.idea/`、`.vscode/`、`*.iml`、`.DS_Store`
- 临时与日志：`*.log`、`*.tmp`、`*.swp`
- 调试材料：`reports/`、`screenshots/`、`docs/e2e-screenshots-*/`
- 测试过程报告：`docs/Browser-MCP-全角色全功能测试报告-*.md`

## .gitignore 对齐说明
当前 `.gitignore` 已覆盖上述 `must not submit` 目录和文件模式（含 `docs/e2e-screenshots-*` 与 `docs/Browser-MCP-*.md`），用于保证最终提交仅包含可运行与答辩所需交付物。
